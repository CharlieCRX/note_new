对应用户配置。

```c

{

    uint32_t channelCount;

    uint32_t channelPeriodMs;
```

# 单通道执行

例如：

```c
execute_channel(ch);
```

内部：

```c
openOneChannel(ch);

ad5932_start_sweep();

while(!ad5932_is_sweep_done())
{
}

closeOneChannel(ch);
```

------

我建议再加：

```c
ad5932_interrupt();
```

或者：

```c
ad5932_set_standby(true);
```

保证DDS处于停止状态。

所以：

```c
void execute_channel(uint8_t ch)
{
    openOneChannel(ch);

    ad5932_start_sweep();

    while(!ad5932_is_sweep_done())
    {
    }

    closeOneChannel(ch);

    ad5932_interrupt();
}
```

# 时间控制

关键来了。

假设：

```text
duration = 2ms
```

------

实际：

```text
actualDuration

=
1987us
```

------

那么：

```text
slot

=
20ms
```

------

执行完：

```text
2ms
```

以后：

```text
sleep

=
18ms
```

------

代码：

```c
uint64_t begin = get_tick_ms();

execute_channel(ch);

uint64_t used =
get_tick_ms()-begin;

if(slotMs > used)
{
    sleep_ms(slotMs-used);
}
```

------

这样：

```text
CH1
始终在

0
200
400
600
...
```

出现。

不会漂移。
