# 📘 单轴物理换算一致性验收报告

------

# 一、验收目的

验证以下内容：

1. PLC 物理位移换算逻辑是否正确
2. 齿轮比参与方向是否正确
3. 导程参与方式是否正确
4. 编码器脉冲与物理位移是否数学一致
5. 是否存在比例误差或模型错误

本阶段不验证机械精度，仅验证数学模型一致性。

------

# 二、机械与参数设定

## 2.1 真实机械关系

电机 10 圈 → 负载 1 圈

属于 10:1 减速关系。

------

## 2.2 PLC 参数设置

| 项目               | 数值             |
| ------------------ | ---------------- |
| 导程 Lead          | 10 mm/rev        |
| 负载齿轮数（分子） | 10               |
| 电机齿轮数（分母） | 1                |
| 编码器分辨率       | 131072 pulse/rev |

------

# 三、理论换算模型

已知：

```
Load_rev = Move_mm / 10
Motor_rev = Load_rev × (10 / 1)
Pulse = Motor_rev × 131072
```

合并后：

```
Pulse = Move_mm × 131072
```

因此理论关系为：

```
1 mm = 131072 pulse
```

# 五、误差分析

观察结果：

1. 误差为固定偏差（≈ +6 pulse）
2. 不随位移增大而放大
3. 不随比例变化而放大
4. 正负方向一致
5. 齿轮比变化后比例正确

由此可判定：

✔ PLC 公式参与方向正确
✔ 导程参与方式正确
✔ 齿轮比参与方式正确
✔ 编码器分辨率参与正确
✔ 系统数学模型一致

------

# 六、误差量级评估

误差 6 pulse 对应：

```
6 / 131072 ≈ 0.0000457 rev
```

因为当前：

```
1 rev = 1 mm
```

所以：

```
误差 ≈ 0.0000457 mm
≈ 0.045 μm
```

结论：

该误差远低于任何实际机械加工精度，属于可忽略范围。

推测来源：

- 伺服内部整数化处理
- 显示刷新周期
- 初始偏移量

属于系统常规量化误差。
