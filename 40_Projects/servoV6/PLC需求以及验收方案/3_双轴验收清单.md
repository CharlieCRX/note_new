# 双轴X1X2验收报告（3.5）

## 一、单轴功能X1、X2 

### 点动运行

首先看轴X1X2点动运行。

配置流程为：

```tex
使能轴X电机: ON
轴X联动使能: OFF
轴X手动速度 = 5.0

轴X1/X2点动正转
轴X1/X2点动反转
```

此时查看轴X1/X2状态：

```tex
Axis_X1/2.bPowerState: ON (同时 Axis_X2.bPowerState: ON)
Axis_X1/2.fActVelocity: 5.01 ~ 4.98（反转 -5.0 ~ -4.98）
```

检测结果：

- X1/X2可以正常点动，且速度与 `轴X手动速度` 相同
- X1/X2使能后，`轴X状态显示`在1和2反复跳转（1：使能 - 空闲 2：正在运动）

不一样的是：

- X1点动：
  - `Axis_X1.fActVelocity`变化
  - `轴X1CurPosition`变化
  - `轴X绝对位置`变化
  - `轴X相对位置`变化
  - `轴X状态显示`固定显示为 2 - 正在运行中
- X2点动：
  - `Axis_X2.fActVelocity`变化
  - `轴X2CurPosition`发生变化
  - `轴X绝对位置`**不**变化
  - `轴X相对位置`**不**变化
  - `轴X状态显示`也固定显示为 2 - 正在运行中
