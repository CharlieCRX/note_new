# 双轴X1X2验收报告（3.5 - 3.6）

## 一、单轴功能X1、X2 

### 点动运行

首先看轴X1X2点动运行。

配置流程为：

```
使能轴X电机: ON
轴X联动使能: OFF
轴X手动速度 = 5.0 mm/s

轴X1/X2点动正转
轴X1/X2点动反转
```

此时查看轴X1/X2状态：

```
Axis_X1/2.bPowerState: ON (同时 Axis_X2.bPowerState: ON)
Axis_X1/2.fActVelocity: 5.01 ~ 4.98（反转 -5.0 ~ -4.98）
```

检测结果：

- X1/X2可以正常点动，且速度与 `轴X手动速度` 相同
- X1/X2使能后，`轴X状态显示`在1和2反复跳转（1：使能 - 空闲 2：正在运动）
- X1/X2位置以物理单位持续变化

不一样的是：

- X1点动：
  - `Axis_X1.fActVelocity`变化
  - `轴X1CurPosition`变化
  - `轴X绝对位置`变化
  - `轴X相对位置`变化
  - `轴X状态显示`固定显示为 2 - 正在运行中
- X2点动：
  - `Axis_X2.fActVelocity`变化
  - `轴X2CurPosition`发生变化
  - `轴X绝对位置`**不**变化
  - `轴X相对位置`**不**变化
  - `轴X状态显示`也固定显示为 2 - 正在运行中

转动方向：

设计X1X2的时候，默认X1和X2是镜像安装组成的龙门架结构。

也就是说，X1的前进应该是跟X2后退的方向一致。我们此时查看X1转动方向：

- X1正向移动：逆时针
- X1负向移动：顺时针

继续观察X2转动方向：

- X2正向移动：逆时针
- X2负向移动：顺时针

可以看出，转动方向是不对的。这样在单轴调整的时候，方向肯定不对了。

### 定位（相对 / 绝对）

问题：X1/X2并没有单独位置移动的选项。

疑问：如果X1/X2电机确实发生了超差，仅凭点动难以高精度移动到目标位置。

### 限位动作

分别在

```
工程管理模块 - 配置 - 运动轴控制 - Axis_X1/X2 - 模式/参数设置
```

配置`软件限位使能`，并配置：

```
负限位： -100mm
正限位： 1000mm
```

- X1移动到负限位后，`轴X1点动反转`自动由`ON`变为`OFF`，此时可以通过`轴X1点动正转`来逆向脱离负限位
- X2移动到负限位后，`轴X2点动反转`自动由`ON`变为`OFF`，此时可以通过`轴X2点动正转`来逆向脱离负限位
- X1移动到正限位后，`轴X1点动正转`自动由`ON`变为`OFF`，也可以正常脱离正限位
- X2移动到正限位后，`轴X2点动正转`自动由`ON`变为`OFF`，也可以正常脱离正限位

问题：

触发到限位后，并没有提示，只是停止了。并能允许反向脱离限位。

### 正向点动同步运动命令

来到了重头戏，开始检测X1X2双轴联动状态下，两部电机的：

- 运动相反：以物理速度设定两个轴相同速度，X1X2方向是相反的（观测`Axis_Xx.fActVelocity`）
- 误差可控：在0负载下，按照最大速度运行，位置同步误差在可接受阈值内。

> 这里的 0 负载最大速度可以参考之前X轴的配置：
>
> 齿轮直径：
>
> ```
> D = 44 mm
> ```
>
> 移动一圈距离：
>
> ```
> L = π × D
> ```
>
> 计算：
>
> ```
> L ≈ 3.1415926 × 44
> ≈ 138.23 mm
> ```
>
> ------
>
> 最大速度：
>
> ```
> 100 mm/s
> ```
>
> 负载轴转速：
>
> ```
> Load_rps = 100 / 138.23
> ≈ 0.723 rps
> ```
>
> ------
>
> 电机转速：
>
> 减速比：
>
> ```
> 30:1
> ```
>
> 所以
>
> ```
> Motor_rps = 0.723 × 30
> ≈ 21.69 rps
> ```
>
> 换算 RPM：
>
> ```
> 21.69 × 60
> ≈ 1301 rpm
> ```

首先按照之前生产条件下，配置X轴的齿轮比和负载：

```
配置Axis_X1/X2的单位换算设置

选中 “使用变速装置”

配置 “工作台旋转1圈的移动量” = 3.1415926 × 44 ≈ 138.23 mm

齿轮比分子（工作台） = 30

齿轮比分母（电机） = 1
```

配置好后，我们按照实际最极限工况的速度 `100mm/s` 来跑。

当双轴联动跑到正限位`1000mm`后，`轴X状态显示`出现了状态`3`：

```
轴X1CurPosition： 1000.003
轴X2CurPosition： -999.99469

轴X状态显示 - 3：报警（报警解除后回到1空闲）
```

但是此时：

```
轴X报警代码 - 0 （无报警）
```

那么到底什么报警了呢？此时无法反向转动和移动。

此时尝试`轴X报警解除触发`，设置为 ON 后，`轴X状态显示`为1、2的正常场景。但是此时仍无法使用点动反转。

此时尝试将X1X2使能和同步全部关闭后，重新打开。此时出现了奇怪的场景：

```
使能轴X电机 ON
轴X联动使能 ON

轴X状态显示 : 3

但是此时：
轴X报警代码： 0（无报警）
```

可以看出，使能并且联动X1X2后，在正限位处，轴X状态显示直接为 3（报警）。那么如何撤离这个报警？

> 按照正常判断，需要手动点动远离这个位置，但是现在无法做到这个功能。
>
> 并且为何触发报警？

尝试单轴移动出正限位继续测试。

```
X1 点动反转 + X2 点动正转

轴X1CurPosition：831.76318
轴X2CurPosition：-809.79468
```

### 反向点动同步运动命令

初始位置：

```
轴X1CurPosition：831.76318
轴X2CurPosition：-809.79468
```

```
轴X联动使能 ON
```

此时`轴X2CurPosition = -831.76318`。

然后反向点动`轴X1点动反转`到负限位，持续检测转速：

```
伺服监视面板X1转速：1302RPM
伺服监视面板X2转速：-1303RPM
```

到达负限位后：`轴X1点动反转`自动跳变为`OFF`，此时无报警。

尝试正向点动撤离负限位到正限位。此时也出现了上文提到的：

```
轴X报警代码：0
轴X状态显示： 3
