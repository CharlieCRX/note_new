# 一、单轴X1、X2运动

## 1.点动运行

### 控制逻辑

首先将X1/X2控制在正负限位内部。

```
使能轴X电机 = ON		（使能X1、X2电机）
轴X手动速度 = 10mm/s	（同步设置X1、X2电机点动速度为 10mm/s）

轴X1/X2点动正转
轴X1/X2点动反转
```

### 状态监测

单独点动需要检测的状态有：

```
轴X状态显示
轴X运动完成
轴X报警代码
轴X点动中
轴X1/X2当前位置
```

这是统一的变化：

- 使能轴X电机 前：
  - 轴X状态显示：0
  - 轴X运动完成：OFF
- 使能轴X电机 后：
  - 轴X状态显示：1 or 2 跳变
  - 轴X运动完成：ON

下面是使能轴X电机后，单独控制X1、X2进行电动时候的状态记录。

### （1）单轴X1、X2点动正常运行状态检测

首先进行：轴X1点动正转，此时状态变化：

- 轴X运动完成：从 ON 到 OFF
- 轴X点动中：从 OFF 到 ON

同样进行X1点动反转，此时状态变化也为：

- 轴X运动完成：从 ON 到 OFF
- 轴X点动中：从 OFF 到 ON

且通过“轴X1当前位置”可以实时监测位置信息。

X2点动后，其状态变化也相同，这里不再赘述。

可以看出X1、X2单独点动控制的时候，都需要通过相同的状态变量来检测电动状态。但是各自的位置信息是区分开的。

### （2）单轴X1点动至正负限位的状态检测

我们在点动的基础上，继续运动，直到正负限位处。

首先查看X1轴设置的正负软限位：

- 轴X1软正极限：1000.000
- 轴X1软负极限：-100.0000

首先将X1点动运行到正极限 1000 mm 处，并观察状态变化：

- 点动运行但是尚未到达正极限之前：
  - 轴X1点动正转：此控制按钮显示 ON
  - 轴X状态显示：2
  - 轴X运动完成：OFF
  - 轴X报警代码：0
  - 轴X点动中：ON
- 点动到到达正极限后：
  - 轴X1点动正转：自动跳变为 OFF
  - 轴X状态显示：3
  - 轴X运动完成：ON
  - 轴X报警代码：3
  - 轴X点动中：OFF

此时通过查看X状态和报警代码含义：

| 4    | 轴X状态显示 | INT  | 不保持 | D100 | 读取： 0：未使能 - 停止 1：使能（未运动） - 空闲  2：使能在运动 - 正在运动 3：报警（报警解除后回到1空闲） |
| ---- | ----------- | ---- | ------ | ---- | ------------------------------------------------------------ |
| 12   | 轴X报警代码 | INT  | 不保持 | D110 | 读取： 0：无报警 1：指令报警 2：伺服报警 3：限位报警         |

可以通过状态显示和报警代码看出轴X1已经运动到限位（正负不区分）。

此时尝试反向点动撤离正极限：

- 轴X状态显示：2
- 轴X运动完成：OFF
- 轴X报警代码：0
- 轴X点动中：ON

可以看出可以直接反向点动撤离正极限。且状态恢复正常了。

我们再看运动到负限位时，轴X的状态：

- 轴X1点动反转：自动跳变为 OFF
- 轴X状态显示：1和2跳变
- 轴X运动完成：ON
- 轴X报警代码：0
- 轴X点动中：OFF

可以看出，此时并没有报警代码提示。且此时可以安全撤离反限位。

### （2）单轴X2点动至正负限位的状态检测

再来查看X2轴设置的正负软限位：

- 轴X2软正极限：100
- 轴X2软负极限：-1000

首先将X2点动运行到正极限 100 mm 处，并观察状态变化：

- 点动运行但是尚未到达正极限之前：
  - 轴X2点动正转：此控制按钮显示 ON
  - 轴X状态显示：2
  - 轴X运动完成：OFF
  - 轴X报警代码：0
  - 轴X点动中：ON
- 点动到到达正极限后：
  - 轴X2点动正转：自动跳变为 OFF
  - 轴X状态显示：2和3跳变
  - 轴X运动完成：ON
  - 轴X报警代码：0到3跳变
  - 轴X点动中：OFF

此时可以通过点动反转安全撤离正极限。

可以看出，到达正限位后，可以安全停下X2。但是到达正限位状态提示存在问题，没法得知是否到达了限位。

我们继续查看X2轴的负限位 -1000 mm 状态逻辑：

- 点动运行但是尚未到达负极限之前：
  - 轴X2点动反转：此控制按钮显示 ON
  - 轴X状态显示：2
  - 轴X运动完成：OFF
  - 轴X报警代码：0
  - 轴X点动中：ON
- 点动到到达负极限后：
  - 轴X2点动反转：自动跳变为 OFF
  - 轴X状态显示：1、2和3跳变
  - 轴X运动完成：ON
  - 轴X报警代码：0到3跳变
  - 轴X点动中：OFF

此时也可以通过点动正转安全撤离负极限。

但是到达限位后的状态提示依旧有问题，无法正确检测是否到达限位。

## 2.绝对定位

现在已经加入了单X1、X2的绝对定位。我们在限位内，进行绝对定位移动。

主要是查看是否位置吻合。

### 控制逻辑

首先将X1控制在正负限位内部，然后设置以下参数：

```
使能轴X电机 = ON
轴X定位速度 = 10 mm/s

轴X1当前位置 = 276.40005
轴X1单独绝对定位距离 = 800.0000
轴X1单独绝对定位触发
```

且设置之前查看电机状态：

- 轴X状态显示：1 or 2 跳变
- 轴X运动完成：ON
- 轴X报警代码：0
- 轴X绝对定位中：OFF

此时将 轴X1单独绝对定位触发 置为 ON 后再置为 OFF，此时运行的时候状态为：

- 轴X状态显示：2
- 轴X运动完成：OFF
- 轴X报警代码：0
- 轴X绝对定位中：ON

运动到目标位置 800 mm 后，提示信息：

- 轴X状态显示：1 or 2跳动
- 轴X运动完成：ON
- 轴X报警代码：0
- 轴X绝对定位中：OFF

由状态变化可以看出，通过状态 “轴X运动完成” 和 “轴X绝对定位中” 可以来检测X1电机的单独定位状态。

但是检测X2的绝对定位触发时候，发现提示信息不对：

- 轴X状态显示：2
- 轴X运动完成：OFF
- 轴X报警代码：0
- 轴X绝对定位中：OFF

可以看出X2的运动状态有缺陷，无法得知X2定位状态和是否完成。

# 二、双轴X1X2运动

## 1.联动前的位置超差检测

当X1、X2位置不互为相反数的时候（不在同一条直线），此时不允许联动。

我们检测下当X1X2在超差的前提下，使用联动会提示什么信息。

首先我们看下位置信息：

```
轴X1当前位置 = 17.92009
轴X2当前位置 = -564.8801
```

可以看出位置是超差的。

然后看下轴X状态信息：

- 轴X状态显示：1 or 2跳变
- 轴X联动状态：OFF
- 轴X报警代码：0
- 轴X软件限位状态：OFF

此时设置 轴X联动使能 为 ON。但是此时：

- 轴X联动状态：OFF

可以看出，确实限制了超差场景。

我们此时在：

- 轴X联动使能 ：ON
- 轴X联动状态：OFF

的超差场景下，进行X单轴的移动

```
轴X1单独绝对定位触发
轴X2单独绝对定位触发

轴X1点动正转
轴X1点动正反转

轴X2点动正转
轴X2点动正反转
```

联动超差后，确实也限制了单轴移动。
