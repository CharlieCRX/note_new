# Bug 汇总

以下为测试中发现的问题。

------

# BUG 1

## X2轴位置未同步系统坐标

### 现象

X2点动时：

```
轴X2CurPosition变化
```

但：

```
轴X绝对位置不变
轴X相对位置不变
```

### 影响

无检测轴2运动位置的标志（暴露给外界的接口）

### 严重级别

**严重**。无法单轴控制其运动。

------

# BUG 2

## 龙门轴方向未做镜像

### 现象

单轴运行的时候：

```
X1正向 = 逆时针
X2正向 = 逆时针
```

### 正确行为

```
X1正向 = X2反向
```

### 严重级别

**严重**

------

# BUG 3

## 无单轴定位功能

X1 / X2 无：

```
绝对定位
相对定位
```

仅有点动。

### 严重级别

**严重**

------

# BUG 4

## 同步启动缺少位置误差检测

系统允许在：

```
X1 ≠ X2
```

时直接开启联动。

### 正确行为

应检查：

```
|X1 + X2| < 阈值
```

否则禁止启动。

### 严重级别

**严重**

------

# BUG 5

## 同步启动强制修改轴位置

开启联动后：

```
X2 = -X1
```

### 风险

可能造成机械冲击。

### 严重级别

**严重**

------

# BUG 6

## 限位触发无报警提示

触发限位：

系统仅停止运动。

没有报警。

### 严重级别

**中等**

------

# BUG 7

## 限位后进入错误报警状态

现象：

双轴移动到正限位后：

```
轴X状态 = 3
报警代码 = 0
```

无法恢复。

### 严重级别

**严重**

------

# BUG 8

## 报警解除逻辑失效

执行：

```
轴X报警解除触发
```

状态恢复：

```
1 / 2
```

但仍无法运动。

### 严重级别

**严重**

------

# BUG 9

## 报警代码系统未正确赋值

出现：

```
状态=报警
报警码=0
```

### 严重级别

**严重**

------

# 三、Bug 严重度统计

| 严重等级 | 数量 |
| -------- | ---- |
| 严重     | 7    |
| 中等     | 1    |

------

# 四、建议优先修复顺序

开发建议按顺序修复：

1️⃣ 坐标系统错误（BUG1）
2️⃣ 龙门方向逻辑（BUG2）
3️⃣ 同步启动安全（BUG4）
4️⃣ 同步强制改坐标（BUG5）
5️⃣ 状态机报警逻辑（BUG7）
6️⃣ 报警系统（BUG9）
7️⃣ 单轴定位功能（BUG3）
