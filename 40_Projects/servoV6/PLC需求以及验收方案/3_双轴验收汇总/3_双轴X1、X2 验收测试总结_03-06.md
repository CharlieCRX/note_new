# 双轴 X1 / X2 验收测试总结

# 1.测试目的

对 PLC 双轴龙门结构（X1/X2）的运动控制功能进行验收测试，重点验证：

- 单轴运动能力
- 双轴同步能力
- 限位安全机制
- 状态机与报警机制
- 机械单位换算正确性

测试按照 PLC 开发需求文档中的功能规范进行。

------

# 2. 总体结论

本次测试结果表明：

**基础运动功能基本可用，但同步控制与状态管理存在多项逻辑问题。**

主要结论如下：

| 模块         | 结论                     |
| ------------ | ------------------------ |
| 单轴点动     | 基本正常                 |
| 单轴速度控制 | 正常                     |
| 位置反馈     | X1正常，X2异常           |
| 软件限位     | 功能有效，但缺乏报警提示 |
| 双轴联动启动 | 存在逻辑缺陷             |
| 双轴同步运动 | 基本可运行               |
| 同步安全机制 | 不完善                   |
| 报警系统     | 存在严重缺陷             |

综合评估：

**当前版本 PLC 程序仍不具备正式生产条件，需要开发进一步修复。**

------

# 3. 单轴功能测试结果

## 3.1 点动运行

配置：

```
使能轴X电机 = ON
轴X联动使能 = OFF
轴X手动速度 = 5 mm/s
```

测试结果：

| 项目     | 结果             |
| -------- | ---------------- |
| X1点动   | 正常             |
| X2点动   | 正常             |
| 实际速度 | 4.98 ~ 5.01 mm/s |
| 方向控制 | 正常             |

说明：

点动功能可正常驱动电机。

------

## 3.2 位置反馈

X1点动时：

| 变量                 | 是否变化 |
| -------------------- | -------- |
| Axis_X1.fActVelocity | ✔        |
| 轴X1CurPosition      | ✔        |
| 轴X绝对位置          | ✔        |
| 轴X相对位置          | ✔        |

X2点动时：

| 变量                 | 是否变化 |
| -------------------- | -------- |
| Axis_X2.fActVelocity | ✔        |
| 轴X2CurPosition      | ✔        |
| 轴X绝对位置          | ❌        |
| 轴X相对位置          | ❌        |

结论：

**X2轴的位置没有同步更新到控制系统的绝对/相对位置。**

PLC 需要开辟轴 X2 单独的位置和速度信息。

------

# 4. 单轴方向问题

龙门结构要求：

```
X1 正向 = X2 反向
```

实际结果：

```
X1 正向：逆时针
X2 正向：逆时针
```

两轴方向完全相同。

结论：

**龙门镜像方向逻辑未实现。**

------

# 5. 单轴定位功能

需求文档要求支持：

- 相对定位
- 绝对定位

实际系统：

**X1/X2 没有单轴定位接口。**

只能点动。

影响：

- 无法精确调整龙门两侧位置
- 超差补偿无法实现

------

# 6. 软件限位测试

配置：

```
负限位 -100 mm
正限位 1000 mm
```

测试结果：

| 功能     | 结果 |
| -------- | ---- |
| 触发限位 | ✔    |
| 停止运动 | ✔    |
| 反向脱离 | ✔    |

但系统行为：

触发限位后

- 没有报警提示
- 没有状态提示

结论：

**限位保护存在，但缺乏报警提示。**

------

# 7. 双轴同步控制测试

## 7.1 联动启动条件

测试场景：

```
X1位置 = 779 mm
X2位置 = 671 mm
```

直接开启联动：

```
轴X联动使能 = ON
```

系统允许启动。

并且：

```
X2位置直接被重置
```

结论：

系统 **没有检测同步误差**。

------

## 7.2 联动启动行为

开启联动后：

```
X2位置 = - X1位置
```

系统强制修改位置。

结论：

**同步逻辑存在强制坐标覆盖行为。**

这是非常危险的行为。

------

# 8. 双轴同步运动测试

在最大工况下测试：

```
速度 = 100 mm/s
```

对应电机：

```
≈1300 RPM
```

结果：

| 项目     | 结果      |
| -------- | --------- |
| X1转速   | 1302 RPM  |
| X2转速   | -1303 RPM |
| 同步方向 | 正常      |

同步运动基本正常。

------

# 9. 限位运行结果

到达正限位：

```
X1 = 1000 mm
X2 = -1000 mm
```

系统状态：

```
轴X状态显示 = 3（报警）
```

但是：

```
报警代码 = 0
```

并且：

- 无法撤离限位
- 无法反向运动

结论：

系统进入 **异常死锁状态**。

------

# 10. 状态机问题

系统定义：

```
1 = 空闲
2 = 运行
3 = 报警
```

实际问题：

```
状态=3
报警码=0
```

出现：

**无报警的报警状态。**

且无法恢复。

------

# 11. 总体评估

当前 PLC 控制程序存在以下主要问题：

- 同步控制逻辑不完善
- 位置管理存在缺陷
- 状态机逻辑错误
- 报警机制不完整

系统仍需开发进一步完善。

------
