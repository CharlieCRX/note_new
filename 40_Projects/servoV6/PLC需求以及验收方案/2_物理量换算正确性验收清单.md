## 📘 第二阶段：物理量换算正确性验收清单（空载专用版）

### 1. 核心逻辑对齐（计算公式）

无论哪种场景，我们要验证的目标都是：**上位机发指令走 $L$ 毫米，电机实际转动 $N$ 圈。**

------

### 场景 A：减速机 + 丝杆（适用于垂直或水平推拉轴）

在这个场景下，需要重点核对 **丝杆螺距（导程）**。

| **步骤** | **验收操作** | **预期结果 (假设减速比 30:1，螺距 10mm)**          |
| -------- | ------------ | -------------------------------------------------- |
| **A-1**  | **参数录入** | 在 AutoShop 界面录入：减速比 **30**, 螺距 **10.0** |
| **A-2**  | **一圈验证** | 上位机指令：绝对定位到 $10\text{mm}$               |
| **A-3**  | **多倍验证** | 上位机指令：绝对定位到 $100\text{mm}$              |
| **A-4**  | **反馈检查** | 查看 Modbus 寄存器（如位置反馈 $D$ 寄存器）        |

------

### 场景 B：减速机 + 齿轮（适用于齿轮齿条或转盘轴）

在这个场景下，没有“螺距”概念，取而代之的是**“齿轮周长”**（即齿轮转一圈，负载移动的距离）。

| **步骤** | **验收操作**      | **预期结果 (假设减速比 30:1，齿轮分度圆周长 62.8mm)**        |
| -------- | ----------------- | ------------------------------------------------------------ |
| **B-1**  | **参数换算**      | 将“工作台旋转一圈的移动量”设置为 **$62.8$**（即 $\pi \times \text{直径}$） |
| **B-2**  | **周长验证**      | 上位机指令：绝对定位到 $62.8\text{mm}$                       |
| **B-3**  | **转矩/速度逻辑** | 给定速度 $62.8\text{mm/s}$                                   |
| **B-4**  | **脉冲匹配**      | 齿轮比的分母是否考虑了齿轮齿数？                             |

## 🛠️ 空载验收的操作小技巧

由于您手头没有机械结构，只有裸电机，建议您这样做来提高验收精度：

1. **“慢速数圈”**：
   - 将运行速度设得很低（比如 $1\text{mm/s}$）。
   - 在电机输出轴（或减速机输出轴，如果有的话）贴上一张白纸条。
   - 执行一个螺距或一个周长的距离，肉眼数它是不是转了“减速比”数值对应的圈数。
2. **利用伺服驱动器面板**：
   - 不用数肉眼，直接看驱动器面板（如汇川 `H05-02` 或类似查看反馈脉冲的参数）。
   - **计算公式**：$\text{反馈脉冲总数} = \frac{\text{目标距离}}{\text{螺距/周长}} \times \text{减速比} \times \text{编码器分辨率}$。
   - 只要这个数对得上，物理换算就绝对没问题。

## ⚠️ 避坑指南（注意事项）

- **齿轮比的“分子”与“分母”**：在 AutoShop 里，有些界面是“齿轮比”，有些是“减速比”。如果是 $30:1$，要确认 30 填在分子还是分母。如果填反了，电机转 1 圈，PLC 会以为走了 $30$ 倍的距离，非常危险。
- **单位设置**：确认 AutoShop 里的单位是 $0.001\text{mm}$ 还是 $0.01\text{mm}$。这直接决定了您在 Modbus Poll 里发送 `100` 代表的是 $1\text{mm}$ 还是 $100\text{mm}$。
