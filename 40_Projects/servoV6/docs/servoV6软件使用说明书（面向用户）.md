# servoV6软件使用说明书（面向用户）

介绍下多轴系统控制软件的软件使用说明。这里仅限于期待中的 UI 界面逻辑。

## 通用按钮

### 控制轴选择按钮

控制轴分为：

- X1X2龙门轴
- YZ线性轴
- R旋转轴

用户可以按下此按钮，选择要控制的轴即可。

### 运动控制按钮

- 前进按钮
- 后退按钮

运动分为前后两个运动。但是前进后退又分为触发的位置移动和按下的点动。两个按钮的控制逻辑：

点动模式下：

- 按下为触发点动运动，松开即可停止点动

位置移动模式：

- 按一下就可触发对应的位置移动。并且屏蔽多次连按的场景。

### 回原点按钮

用户按下此按钮后，当前轴会执行回机械零点并重新配置绝对零点。

## 单轴YZ界面

现在介绍下，选择Y、Z轴的时候，界面需要展示的信息和提供给用户的操作。

### 1.轴当前状态

- 当前位置信息 mm

- 当前速度信息：mm/s

- 当前运动状态

  ```
  未使能
  空闲
  运动
  报警
  ```

- 报警状态

### 2.轴运动模式

显示：

```
Jog
Position Move
	- relative move
	- absolute move
```

否则用户不知道：

```
当前控制模式
```

单轴支持点动和位置移动。且位置移动包括 相对距离移动 和 绝对位置定位。

点动模式：

- 设置的点动速度

定位移动（位置移动）：

- 设置的定位速度

定位包括相对距离移动和绝对位置定位：

- 设置的绝对位置
- 设置的相对距离

两个定位统一使用设置的定位速度作为位置移动的速度。

### 3.限位信息

显示：

```
软正极限
软负极限
```

同时显示：

```
当前位置在范围内
```

### 4.报警信息

显示：

```
报警代码
报警类型
```

例如：

```
Servo Alarm
Command Alarm
```

### 5.急停按钮

此按钮终止一切进行的操作。

## 联动龙门轴界面

X1X2组成了龙门轴，可以进行同步前进后退运动。此部分讲解X1X2同步模式下，界面要展示的内容。

龙门轴必须增加：

```
同步状态
```

否则操作员会不放心。

### 1.需要多展示的部分

- X1X2联动状态：

  ```
  Linked
  Unlinked
  ```

  并且此按钮需要管理员密码授权才能切换。并且默认是同步状态。

- X1X2位置信息：分别展示主轴X1和从轴X2的位置信息 mm

- 当前同步误差：Error = |X1 - X2|

- 当前运动状态

  ```
  未使能
  空闲
  运动
  报警
  ```

- 报警状态（这里需要加入超差报警状态）

其他逻辑与YZ轴类似不予赘述。

### 2.位置超差报警

当出现了位置超差大于设定的阈值，就要给用户提示超差报警。此时需要用户手动解除X1X2联动，手动调节X1X2进行补偿。

## 位置超差时候的X1或X2界面

本质与YZ相同。

只是逻辑不同。
