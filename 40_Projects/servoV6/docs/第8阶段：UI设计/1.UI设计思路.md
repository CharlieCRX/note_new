## 一、行车控制软件的功能

行车控制软件可以控制基于 PLC 控制的多组电机进行运动和信息修改以及展示。

### A. 运动模式切换

支持行车进行：

- 点动：并且可以在点动模式下，设置点动速度。
- 定位移动

在点动模式下，可以进行的操作：

- 设置点动速度
- 点动控制前进/后退（按钮展示控制）

定位模式下，可以进行的操作：

- 设置定位速度（这个速度就是绝对位置和相对位置移动的速度）
- 进一步切换运动模式：
  - 绝对位置移动：可以设置绝对位置移动坐标
  - 相对位置移动：可以设置相对位置移动距离（支持正负选择）
- 统一的执行按钮，可以通过按下执行，来触发相对和绝对位置移动。

### B. 电机状态展示

展示电机信息有：

- 电机运动状态：与`Axis`中定义的`AxisState`相关
- 电机的位置信息
  - 绝对位置信息
  - 相对于相对原点的位置信息
- 电机当前位置以及限位信息的状态展示

### C. 组/设备切换

这个模块负责：

- 切换分组：类似GroupA、GroupB
- 展示多轴：YZR、X龙门轴

## 二、UI展示设备

`servoV6`的展示环境有两个：

- 桌面的 1080P 显示器（27寸），部署平台为 windows 环境
- 掌机（Android 环境）：
  - 屏幕尺寸 153mm x 86 mm
  - 分辨率：1920 * 1080
  - 主屏对角线：6.9英寸

我们是跨系统控制软件，所以采用 QML + C++ 后端的架构设计项目。

所以基于掌机这种屏幕较小的设备，要保证工业控制的稳定性以及用户的交互优良性。
