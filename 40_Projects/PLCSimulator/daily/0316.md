------

# 一、推荐方案（最简单稳定）

直接在 **Linux 虚拟机里开发**。

环境：

```
Ubuntu / Debian
gcc
cmake
make
gdb
git
```

这是 **Linux 工程师标准环境**。

工业控制软件很多都是：

```
Linux + C++
```

------

# 二、安装 C++ 编译环境

在 Linux 终端执行：

```bash
sudo apt update
sudo apt install build-essential cmake gdb git
```

安装内容：

| 工具      | 作用      |
| --------- | --------- |
| gcc / g++ | C++编译器 |
| make      | 构建工具  |
| cmake     | 项目构建  |
| gdb       | 调试      |
| git       | 版本管理  |

安装完成后验证：

```bash
g++ --version
cmake --version
gdb --version
```

如果输出版本号说明成功。

------

# 三、创建你的第一个项目

创建目录：

```bash
mkdir plc_simulator
cd plc_simulator
```

创建结构：

```bash
mkdir src
mkdir tests
```

------

# 四、创建第一个 C++ 文件

创建：

```
src/main.cpp
```

写入：

```cpp
#include <iostream>

int main()
{
    std::cout << "PLC Simulator Start\n";
    return 0;
}
```

------

# 五、写 CMakeLists.txt

在项目根目录创建：

```
CMakeLists.txt
```

内容：

```cmake
cmake_minimum_required(VERSION 3.10)

project(plc_simulator)

set(CMAKE_CXX_STANDARD 17)

add_executable(plc_simulator
    src/main.cpp
)
```

------

# 六、编译项目

执行：

```bash
mkdir build
cd build

cmake ..
make
```

运行：

```bash
./plc_simulator
```

输出：

```
PLC Simulator Start
```

说明 **环境成功**。
