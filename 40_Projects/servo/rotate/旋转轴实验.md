# 旋转轴角度一致性验证实验设计报告

## 1. 实验背景与目的

### 1.1 背景说明

当前旋转轴系统由以下部分组成：

- 伺服电机侧：
  配置伺服自带**绝对位置编码器**，用于获取电机轴的绝对角度信息；
- 负载侧（旋转轴）：
  安装**外置绝对位置编码器**，用于获取旋转轴的真实机械角度；
- 中间传动机构：
  采用**蜗轮蜗杆传动结构**。

在系统调试过程中发现：

- 在部分角度区间及不同旋转方向下，
  电机角度与负载侧角度之间存在可观测偏差；
- 偏差在往返运动后可消除，但在单方向运动中表现明显；
- 在 360° 过零点附近，角度差异表现出不连续特性。

### 1.2 实验目的

本实验的目标是：

1. **验证旋转轴系统中角度偏差的来源是否为蜗轮蜗杆传动的非线性与方向相关特性**；
2. 判断该偏差是否具有：
   - 确定性
   - 可重复性
   - 可建模性；
3. 为后续是否需要引入**传动映射补偿（LUT / 函数建模）**提供实验依据；
4. 为实现 **全 360° 角度偏差 ≤ 0.1°** 的工程目标提供数据支持。

------

## 2. 实验基本原则与约束条件

### 2.1 基本假设

- 伺服侧绝对编码器角度为**真实电机轴角度**；
- 外置绝对编码器角度为**真实负载轴角度**；
- 两类编码器均不存在丢计数或随机测量误差；
- 系统角度偏差仅可能来源于：
  - 蜗轮蜗杆传动特性
  - 机械弹性与方向迟滞。

### 2.2 实验统一要求

- 所有角度数据需转换为**连续角度域（unwrap）**进行分析；
- 实验过程中避免频繁正反切换，除非实验目的明确要求；
  - 频繁正反切换会把“确定性的传动特性”，变成“不可重复的混合状态”，从而直接破坏实验结论的可信度。
- 每次实验仅验证一个物理假设，避免变量耦合。

------

## 3. 实验项目设计

------

## 实验一：单方向全行程传动线性验证

### 3.1 实验目的

验证蜗轮蜗杆传动中，**电机轴角度与负载轴角度之间是否存在线性比例关系**。

### 3.2 实验方法

- 旋转方向：仅顺时针（CW）；
- 起始位置：任意角度；
- 连续旋转角度：≥ 450°（确保完全加载传动）；
- 采样间隔：初步采用 10°；
- 记录数据：
  - 电机角度 `θ_motor`
  - 负载角度 `θ_load`。

### 3.3 数据分析方式

绘制关系曲线：

- 横轴：`θ_motor`
- 纵轴：`θ_load`

### 3.4 判定标准

- 若曲线明显偏离直线或斜率存在周期性变化，
  则可判定：

  > 传动比为角度相关的非线性函数，而非常数。

------

## 实验二：同角度区间的方向一致性验证

### 4.1 实验目的

验证蜗轮蜗杆传动是否具有**方向相关特性**。

### 4.2 实验方法

1. 顺时针旋转至指定角度 θ₁；
2. 记录此时负载角度 θ_load(CW)；
3. 逆时针旋转至相同电机角度 θ₁；
4. 再次记录负载角度 θ_load(CCW)。

### 4.3 数据分析方式

计算方向差异：

```C
Δθ = θ_load(CW) − θ_load(CCW)
```

### 4.4 判定标准

- 若 Δθ ≠ 0 且具有稳定重复性，
  则确认传动关系具有明显方向相关性。

------

## 实验三：微小反向迟滞验证（死区测试）

### 5.1 实验目的

验证蜗轮蜗杆传动中是否存在**反向迟滞或角度死区**。

### 5.2 实验方法

1. 顺时针加载至指定角度并保持；
2. 逆时针分别微动：
   - 0.1°
   - 0.2°
   - 0.5°
   - 1.0°
3. 每一步记录负载角度变化。

### 5.3 判定标准

- 若在一定反向角度内，负载角度基本不变，
  则说明系统存在机械迟滞区；
- 该特性将直接影响高精度定位与补偿策略。

------

## 实验四：360° 闭环一致性验证

### 6.1 实验目的

验证系统是否存在**累计误差或打滑现象**。

### 6.2 实验方法

- 顺时针连续旋转 N 圈（N = 1 / 3 / 5）；
- 回到起始角度；
- 比较起点与终点负载角度。

### 6.3 判定标准

- 若终点角度与起点角度差值 ≤ 0.05°，
  则系统为**确定性系统**，具备建模与补偿基础。

------

## 实验五：局部高分辨率扫描（精度评估）

### 7.1 实验目的

评估在关键角度区间（如 0° 附近）
建立补偿模型所需的最小采样间隔。

### 7.2 实验方法

- 固定单一方向；
- 在目标区间内：
  - 以 0.2° 电机角度为步进扫描；
- 记录负载角度变化。

### 7.3 判定标准

- 若负载角度变化平滑：
  - 允许使用 0.5° 采样点；
- 若存在明显局部起伏：
  - 需提高采样密度（≤ 0.25°）。

------

## 实验六：重复性与稳定性验证

### 8.1 实验目的

验证角度偏差是否具备长期稳定性。

### 8.2 实验方法

- 重复执行实验一与实验二；
- 在不同时间点、不同加载状态下进行；
- 对比误差曲线。

### 8.3 判定标准

- 若误差曲线高度一致：
  - 系统偏差可通过建模补偿；
- 若差异显著：
  - 需重新评估机械设计精度。

------

## 9. 预期结论与工程意义

通过上述实验，可明确：

- 当前系统角度偏差是否来源于蜗轮蜗杆的确定性传动非线性；
- 是否需要引入方向相关的角度映射模型；
- 系统在理论上是否具备实现
  **全 360° 角度偏差 ≤ 0.1°** 的工程可行性。

该实验结果将直接作为后续：

- 角度补偿算法设计
- 标定流程制定
- 系统精度评估与验收的依据。

------
