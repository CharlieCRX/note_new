## 软元件

使用 ModbusTCP 来使能轴Y。给的地址为软元件地址：

| 序号 | 变量名      | 数据类型 | 掉电保持 | 软元件地址 | 解释             |
| ---- | ----------- | -------- | -------- | ---------- | ---------------- |
| 2    | 使能轴Y电机 | BOOL     | 不保持   | M1         | 读写：1：使能轴Y |

那么软元件是什么，并且其地址是什么呢？

> 软元件本质是 PLC 内部内存的抽象命名方式。
>
> 地址 = 这块内存在 PLC 里的编号。

通过阅读手册后，可以得知变量“  使能轴Y电机”是自定义的变量，而其软元件地址映射为了 M1。意味着：

### 地址映射

当在程序里写：

```
使能轴Y电机 := TRUE;
```

实际上 PLC 做的是：

```
M1 = 1
```

当你监控 M1 变成 1 时：

变量“使能轴Y电机”也会显示 TRUE。

它们不是两个东西。

是：

```
一个内存地址
两个名字
```

### 内存大小

M开头代表的是位软元件。长度：1 bit。
