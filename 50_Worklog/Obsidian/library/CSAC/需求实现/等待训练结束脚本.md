在脚本开始时记录开始时间，然后每次执行 `ruclock --update-training` 时，把输出保存到变量中。

如果检测到输出中包含 **`训练已完成`**，就计算运行时间并退出循环。

修改后的脚本如下：

```bash
#!/bin/bash

echo "开始循环读取 ruclock，按 Ctrl+C 停止..."

# 记录开始时间（Unix 时间戳）
start_time=$(date +%s)

while true; do
    echo "===== $(date '+%Y-%m-%d %H:%M:%S') ====="

    # 保存命令输出
    output=$(ruclock --update-training)

    # 输出原始内容
    echo "$output"
    echo ""

    # 检查是否训练完成
    if echo "$output" | grep -q "训练已完成"; then
        end_time=$(date +%s)
        elapsed=$((end_time - start_time))

        echo "========================================"
        echo "训练已完成！"
        echo "总运行时间：${elapsed} 秒"
        echo "========================================"

        break
    fi

    sleep 2
done
```
