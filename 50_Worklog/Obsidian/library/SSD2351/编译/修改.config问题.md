添加了AK5538 codec 驱动后，需要在

```bash
/home/crx/ssd2351/source/kernel
```

中执行

```bash
make menuconfig
make savedefconfig
cp defconfig arch/arm/configs/pcupid_ssm001c_s01a_spinand_voip_defconfig
```

执行完毕后，检查`.config`文件，搜索 5538 关键词：

```bash
CONFIG_SND_SOC_AK5538=y
```

可以看到已经配置了 AK5538 的驱动。

删除无效的文件树：

```bash
rm -f arch/arm64/boot/dts/sstar/pcupid-ssm001c-s01a-voip.dtb
```

然后回到`project/`目录：

```bash
/home/crx/ssd2351/source/project
```

需要执行：

```bash
make image-fast-nocheck -j8
```

来快速打包（不要使用`make image`，不知道为何会修改`.config`）
